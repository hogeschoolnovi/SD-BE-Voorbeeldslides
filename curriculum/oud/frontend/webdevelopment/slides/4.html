JS-objecten (1)

  • Dit zijn altijd objecten:
    • Date
    • Math
    • Regular expression
    • Array
    • Function
    • Object

JS-Objecten (2)

  • Dit zijn soms objecten:
    • Boolean (met new gedefinieerd)
    • Number (met new gedefinieerd)
    • String (met new gedefinieerd)
  • Dit zijn nooit objecten:
    • Primitieve datatypes
    • string, number, boolean, null en undefined

Definieer de waardes NIET met new. Dit zorgt voor extra overhead.

De values (waardes) zijn immutable. Je kunt de waarde van de variabele wel veranderen, maar niet de waarde van de waarde.

Simpel JS-object


                    let student = {
                        firstname : "Nick",
                        lastname : "Stuivenberg",
                        birthYear: 1990,
                        isRegistered : false
                        fullname : function() {
                            return this.firstname + " " + this.lastname;
                    }
                    console.log(student.fullname());

JS-Object met constructor & functies


                    function Book(author, title, isbn, year) {
                        this.author = author;
                        this.title = title;
                        this.isbn = isbn;
                        this.year = year;
                        this.text = function() {
                            return "this book was published in " + this.year;
                        };
                    }
                    let favouriteBook = new Book("JK Rowling", "Harry Potter",
                        "isbn-2-222", 1995);

JS Prototypes (2)

Daarom hebben we prototypes


                    function Teacher(fn, ln) {
                        this.firstname = fn;
                        this.lastname = ln;
                    }
                    // Voeg property toe.
                    Teacher.prototype.subject = "Java";

                    // Voeg functies toe
                    Teacher.prototype.name = function () {
                        return this.firstname + " " + this.lastname;
                    };

JSON

  • Info
  • Van JS object naar JSON
  • Voorbeelden

JSON info

  • Over het web wordt onderling gecommuniceerd met tekst;
  • JSON is in het leven geroepen om data uit te wisselen;
  • Gebaseerd op het JS object. Staat voor Javascript Object Notation
  • Meerdere (bijna alle) talen kunnen overweg met JSON
  • JSON is makkelijk te lezen en zelf beschrijvend

JSON Syntax

  • Data is name/value pairs
  • Data is gescheiden door komma's
  • Curly braces {} voor objecten
  • Blokhaken voor arrays

                        repairmen = {
                            "name":"John",
                            "age":30,
                            "cars": [
                            { "name":"Ford", "models":
                                ["Fiesta", "Focus", "Mustang" ]},
                            { "name":"BMW", "models":
                                ["320", "X3", "X5" ]},
                            { "name":"Fiat", "models":
                                ["500", "Panda" ] }
                            ]
                        }

JSON <-> JS-object

JS naar JSON


                            let jsonString =
                                JSON.stringify(fifaSpeler);

JSON naar JS


                            let jsonObject =
                                JSON.parse(jsonString);

Verschil JSON en JS

  • JSON heeft geen functies*
  • JSON heeft geen date object
  • JSON kent geen undefined
  • In JS mag je single-quotes gebruiken in JSON niet

Cookies

  • Wat zijn cookies?
  • Voorbeeldcode

Cookies

  • Cookies zijn data als kleine tekstbestanden opgeslagen;
  • Cookies zijn uitgevonden om data te onthouden over de bezoeker;
  • Cookies maken ook weer gebruik van een name-value pair.

Cookie maken


                    document.cookie = "teacher=Nick Stuivenberg";

Cookie maken met verwijder datum


                    document.cookie = "teacher=Nick Stuivenberg;
                        expires=Thu, 18 Dec 2025 12:00:00 UTC";

Cookie maken met path parameter


                    document.cookie = "teacher=Nick Stuivenberg;
                        expires=Thu, 18 Dec 2025 12:00:00 UTC;
                        path=/";

Cookie lezen


                    let allCookies = document.cookie;

Cookie aanpassen


                    document.cookie = "teacher=Nova bij Novi;
                        expires=Thu, 18 Dec 2025 12:00:00 UTC";

Cookie verwijderen


                    document.cookie = "teacher=;
                        expires=Thu, 01 Jan 1970 00:00:00 UTC;
                        path=/;";

document.cookie will return all cookies in one string much like: cookie1=value; cookie2=value; cookie3=value;

You should define the cookie path to ensure that you delete the right cookie. Some browsers will not let you delete a cookie if you don't specify the path.

Cookie voorbeeld


                    document.cookie = "teacher=Nick Stuivenberg";
                    document.cookie = "TA=Fred Flatbeklimmer";
                    document.cookie = "TA=Henny Heetijzer";
                    alert(document.cookie);
