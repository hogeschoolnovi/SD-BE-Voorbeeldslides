HTML - terugblik

  • HTML-elementen bestaan uit tags
  • Attributen staan alleen in de start-tag
  • Alle tags worden gesloten*
  • Styling gebeurt in een CSS-bestand

                            <img src="img/formatting.png"

Box model

Block, inline-block & inline

  • display: block
  • display: inline-block
  • display: inline

Voorbeeld - opzet


                        <body>
                        <span class="ex">Hello world.</span>
                        <span class="ex">Hello world.</span>
                        <span class="ex">Hello world.</span>
                        <span class="ex">Hello world.</span>
                        </body>
                        

                        .ex {
                            display: inline;
                            background-color:yellow;
                            height:100px;
                            width:100px;
                            border: 1px solid #f00;
                        }

Voorbeeld - inline

Een inline-element maakt geen gebruikt van height en width. Ook werken de margin-top, margin-bottom en padding-top en padding-bottom niet.

Voorbeeld - inline-block

Er wordt nu wel gebruik gemaakt van de properties:
  • height
  • width
  • margin top & bottom
  • padding top & bottom

Voorbeeld - block

Er wordt nog steeds gebruik gemaakt van properties.
  • Na elk element wordt een line-break toegevoegd.

Pseudo-class


                selector:pseudo-class {
                  property: value;
                }
                

                    /* onbezochte link */
                    a:link {
                      color: #ff0000;
                    }
                    /* visited link */
                    a:visited {
                      color: #00ff00;
                    }
                    /* mouse-over (hover) link */
                    a:hover {
                      color: #ff00ff;
                    }
                    /* Geselecteerde link */
                    a:active {
                      color: #0000ff;
                    }
Je kunt het herkennen aan de dubbele punt

Pseudo-element


                selector::pseudo-element {
                    property: value;
                 }
                

                        p::first-letter {
                            color: #0000ff;
                            font-size: xx-large;
                        }
Je kunt dit herkennen aan twee dubbele punten.

Viewport (1)

Viewport (2)


                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • width=device-width zorgt ervoor dat de breedte van het scherm wordt gevolgd.
  • initial-scale=1.0 zet de initiële zoom.
Dit is een meta-tag, dus waar moet deze in de HTML-code geplaatst worden?
  • width=device-width zorgt ervoor dat de breedte van het scherm wordt gevolgd.
  • initial-scale=1.0 zet de initiële zoom.

Media queries


                    @media only screen and (max-width: 600px) {
                        body {
                            background-color: lightblue;
                        }
                        div.example {
                            display: none;
                        }
                    }
                    @media only screen and (min-width: 600px) {
                        div.example {
                            font-size: 80px;
                        }
                    }
                    @media only screen and (min-width: 768px) {
                        div.example {
                            font-size: 120px;
                        }
                    }
