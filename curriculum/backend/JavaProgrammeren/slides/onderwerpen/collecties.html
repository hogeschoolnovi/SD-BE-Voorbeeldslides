Array - Gebruiksaanwijzing

  • Een Array heeft een vaste grootte. Deze kun je niet uitbreiden
  • Bij instantiatie moet je of de grootte meegeven of de array direct vullen.
  • Je kunt een Array maken van primitieve datatypen en Objecten.
  • Meest voorkomende exceptie: ArrayIndexOutOfBoundsException

Array - uitprinten


            System.out.println(playerNamesTeamTwo[0]);

Je geeft de positie aan tussen de blokhaken.

Array - aanpassen


            playerNamesTeamTwo[0] = "New Player Frank";

Je geeft de positie aan tussen de blokhaken.

Array - Lengte opvragen


            System.out.println(playerNamesTeamTwo.length);

Print de waarde van de Array uit.

Array - ALle waardes uitprinten met foreach


            for(String name : playerNamesTeamTwo) {
                System.out.println(name);
            }

ArrayList

Wat nou als je van tevoren niet de grote weet? Je wilt dus een uitbreidbare lijst hebben.

ArrayList to the Rescue!

ArrayList - voorbeeld met toevoegen


            List<String> playerNames = new ArrayList<>();
            playerNames.add("Player Uno");
            playerNames.add("Player Duo");
            playerNames.add("Player Tres");

Neem deze code over in IntelliJ. Wat gebeurt er?

ArrayList - packages

  • De code om met ArrayListen te werken, moet je importen.
  • Die doe je met Alt + Enter.

            import java.util.ArrayList;
            import java.util.List;

ArrayList - Mogelijkheden


            System.out.println("Eerste positie: " + playerNames.get(0));
            System.out.println("Grootte: " + playerNames.size());
            playerNames.remove(0); // Verwijderd Player Uno
            playerNames.set(1, "Ik ben aangepast"); // Past de waarde op positie 1 aan.

ArrayList - Loopen met een For-lus


            for(int i = 0; i < playerNames.size(); i++) {
                System.out.println("Positie: " + i);
                System.out.println("Naam: " + playerNames.get(i));
            }

In IntelliJ: type fori en druk op Enter

Verschillen

  • Weet je van tevoren de grote niet? ArrayList;
  • ArrayList werkt niet met primitieve datatypen;
  • Vast aantal? Array!

Good to know

  • Java garandeert geen volgorde;
  • Beide mogen null-waardes bevatten;
  • Beide mogen dubbele waardes bevatten.
  • Meerdere collecties: HashMap, LinkedList et cetera.
  • Een Array is altijd sneller dan een ArrayList!
