Methodes

Een methode is een blok code in Java die alleen draait, wanneer deze aangeroepen wordt. Met een methode kun je een stukje code definiëren, die je maar één keer hoeft te schrijven en meerdere keren kan aanroepen.

-EdHub (2020)

Methodes

  • Een methode heeft een naam (beginnend met een kleine letter)
  • Een methode kan parameters ontvangen.
  • Een methode kan 'iets' teruggeven.
  • Een methode kan bovenstaande combineren.
  • Wanneer je een methode aanroept en een waarde meegeeft, zijn die waardes argumenten.

Codevoorbeelden

Let op. Alle methodes worden direct vanuit de main-methode aangeroepen. De main-methode is een static methode. Een static-methode mag alleen andere static methodes aanroepen. Vandaar dat je overal static ziet.

Methodes - De simpelste


            public class Main {
                public static void main(String[] args) {
                    study();
                }
                public static void study() {
                    System.out.println("I study.");
                }
            }

Methodes - Let op

Vanaf nu laten de slides alleen het aanroepen en de methode zien. Dus geen klasse- en packagenamen meer.

Methodes - Input parameters


        public static void main(String[] args) {
            doubleAndPrint(10);
        }
        public static void doubleAndPrint(int number) {
            int result = number * 2;
            System.out.println("The result is:" + result);
        }

Methodes - return


        public static void main(String[] args) {
            int dayOfMonth = getDayOfCurrentMonth();
            System.out.println("Het is vandaag de " + dayOfMonth);
        }
        public static int getDayOfCurrentMonth() {
            int currentDayOfMonth =
                Calendar.getInstance().get(Calendar.DAY_OF_MONTH);
            return currentDayOfMonth;
        }

void verander je in het datatype dat je teruggeeft!

Methodes - Return & parameters


        public static void main(String[] args) {
            String name = "nick";
            String nameWithCapitalizedFirstLetter = capitalizeFirstLetter(name);
            System.out.println("Naam met hoofdletter: "
                + nameWithCapitalizedFirstLetter);
        }
        public static String capitalizeFirstLetter(String uncapitalizedName) {
            String capitalizedName = uncapitalizedName.substring(0, 1).toUpperCase()
                                + uncapitalizedName.substring(1);
            return capitalizedName;
        }
