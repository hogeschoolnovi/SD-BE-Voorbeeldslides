Wat is een variabele?

Een variabele is een naam die wordt gegeven aan een deel van het geheugen, waarin ook een waarde staat. Op deze manier hoef je niet steeds weer een waarde aan te maken, maar kan je het met een naam ophalen uit het geheugen. De waarde kan tijdens het uitvoeren van een programma ook veranderen.

https://programmeerplaats.nl/java-variables/#:~:text=Een%20(Java)%20variable%20(variabele,waarin%20ook%20een%20waarde%20staat.&text=De%20waarde%20kan%20tijdens%20het%20uitvoeren%20van%20een%20programma%20ook%20veranderen.

Wat is een variabele?

Bestaat uit drie onderdelen:

  • Datatype
  • Naam
  • Waarde

Wat is een datatype?

Met een datatype geef je het soort data aan dat je kunt opslaan in een variabele.

Wat is een datatype?

Met een datatype geef je het soort data aan dat je kunt opslaan in een variabele.

  • Integer
  • Double
  • String
  • Boolean

Wat is een datatype?

Met een datatype geef je het soort data aan dat je kunt opslaan in een variabele.

  • Integer: geheel getal
  • Double: getal met cijfer achter de komma
  • String: Een reeks met tekens tussen ""
  • Boolean: true of false
