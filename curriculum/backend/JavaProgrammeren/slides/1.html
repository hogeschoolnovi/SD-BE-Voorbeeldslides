Klasse

  • Java is objectgeoriënteerd
  • Klasse is aan blauwprint om een object te maken
  • Een klasse weet dingen (Instance Variables)
  • Een klasse doet dingen (methods)
  • OO, dus ding of entiteit is het uitgangspunt

Klasse - voorbeeld


                    public class Student {
                        private int studentNumber;
                        private String firstName;
                        private String lastName;
                        private String emailAddress;

                        public void study() {
                            System.out.println("I study.");
                        }
                    }

Klasse - Guidelines

  • Klassenaam is dezelfde als bestandsnaam
  • Klassenaam begint met een hoofdletter
  • Instance variable-namen beginnen met een kleine letter
  • Methodes beginnen met een kleine letter

Objecten

  • Je gebruikt een klasse om een Object te creëren.
  • Een object wordt gemaakt met het new keyword
  • Methodes uit een klasse worden aangeroepn met objectnaam (niet klassenaam).methodenaam()

Objecten - voorbeeld


                public class MainObjectStudent {
                    public static void main(String[] args) {
                        Student nick = new Student();
                        nick.study();
                    }
                }

Methodes

Een methode is een blok code in Java die alleen draait, wanneer deze aangeroepen wordt. Met een methode kun je een stukje code definiëren, die je maar één keer hoeft te schrijven en meerdere keren kan aanroepen.

-EdHub (2020)

Methodes

  • Een methode heeft een naam (beginnend met een kleine letter)
  • Een methode kan parameters ontvangen.
  • Een methode kan 'iets' teruggeven.
  • Een methode kan bovenstaande combineren.

Methodes - De simpelste


                public void study() {
                    System.out.println("I study.");
                }

Methodes - Input parameters


                public void doubleAndPrint(int number) {
                    int result = number * 2;
                    System.out.println("The result is:" + result);
                }
                

                public class MainObjectStudent {
                    public static void main(String[] args) {
                        Student nick = new Student();
                        nick.doubleAndPrint(10);
                    }
                }

Methodes - return


                public int returnDayOfMonth() {
                    int currentDayOfMonth =
                        Calendar.getInstance().get(Calendar.DAY_OF_MONTH);
                    return currentDayOfMonth;
                }
                

                public class MainObjectStudent {
                    public static void main(String[] args) {
                        Student nick = new Student();
                        int dayNumber = nick.findDayOfMonth();
                        System.out.println("Today is the " + dayNumber + "th.");
                    }
                }

Getters & Setters

  • Je wilt instance variables op private hebben zodat je de toegang kunt beheren.
  • Om vanuit andere klasse toch bij de waardes te komen gebruiken we Getters & Setters

Constructors

  • Een object wordt altijd gemaakt met een Constructor
  • Hiervoor wordt het new keyword gebruikt.
  • De naam van de Constructor is dezelfde als de naam van de klasse.
  • Wanneer jij geen constructor toevoegt, voegt Java een lege constructor toe.

ArrayList - Het begin


                import java.util.ArrayList;
                import java.util.List;
                public class MainListExample {
                    public static void main(String[] args) {
                        List<String> carBrands = new ArrayList<>();
                    }
                }

ArrayList - Toevoegen


                List<String> carBrands = new ArrayList<>();
                carBrands.add("Volkswagen");
                carBrands.add("DAF");
                carBrands.add("Tesla");
                carBrands.add("Fiat");

ArrayList - Lezen, updaten en verwijderen


                List<String> carBrands = new ArrayList<>();
                carBrands.add("Volkswagen");
                System.out.println(carBrands.get(0));
                carBrands.set(0, "Porsche");
                carBrands.remove(0);

ArrayList - Laatste


                List<String> carBrands = new ArrayList<>();
                carBrands.add("Volkswagen");
                System.out.println("De grootte: " + carBrands.size());
                carBrands.clear();
                System.out.println("De grootte: " + carBrands.size());
