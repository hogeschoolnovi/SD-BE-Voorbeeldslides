Wat is een database?

Een database is een georganiseerde collectie van gestructureerde informatie (of data) dat is opgeslagen in een (elektronisch) systeem. Elektronisch wordt de data gecontroleerd door een database management systeem (DBMS). Deze twee samen zijn onderdeel van het database-systeem of kortgezegd de database.

https://www.oracle.com/database/what-is-database/

Waar moet de data aan voldoen?

Dat de data

  • (eenvoudig) opslaanbaar zijn;
  • uitgelezen en doorzocht kunnen worden;
  • bewerkt moeten kunnen worden;
  • verwijderd kunnen worden zonder nadelige gevolgen.

Waar moet de data en de database aan voldoen?

  • De data moeten consistent zijn.
    • Sla de data niet dubbel op
    • Relaties moeten blijven kloppen wanneer de gegevens verwijderd worden
  • Autorisatie/Authenticatie
    • Geef niet iedereen toegang tot lezen, schrijven en verwijderen.

    • De database verandert alleen in valide states. Dus een aanpassing maakt de database niet kapot

Database Management Systeem

Een DBMS is software voor het opslaan en verkrijgen van data, waar het in acht nemen verschillende beveiligingskwesties mogelijk zijn.
