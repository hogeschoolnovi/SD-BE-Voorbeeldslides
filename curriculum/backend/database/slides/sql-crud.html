Wat is CRUD?

  • CREATE: INSERT
  • READ: SELECT
  • UPDATE: UPDATE
  • DELETE: DELETE

Wat is SQL

  • SQL is een Data Manipulation Language.
  • Verschillende dialecten. In de basis hetzelfde.
  • Daarmee kun je CRUD-operaties op je dataset doen.
