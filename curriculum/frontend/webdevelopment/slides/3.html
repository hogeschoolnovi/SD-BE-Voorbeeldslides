Waar kun je CSS plaatsen?

Als attribuut van het HTML-element. Dit heet Inline CSS


                    <p style="color:#000; font-family: 'Lato', sans-serif;">
                        Hallo, hallo, hallo
                    </p>

Waar kun je CSS plaatsen?

Binnen de style tags. Dit heet Internal CSS


                    <html>
                    <head>
                        <style>
                            p {
                                font-weight: bold;
                                font-family: "Lato", sans-serif;
                                color:#f00;
                            }
                        </style>
                    </head>
                    <body>
                    <p>Hello</p>
                    </body>
                    </html>

Waar kun je CSS plaatsen?

In een apart CSS-bestand. Dit External CSS


                    <html>
                    <head>
                        <title>Externe CSS</title>
                        <link rel="stylesheet" type="text/css" href="mystyle.css">
                    </head>
                    <body>
                    <h1>Fancy Website</h1>
                    </body>
                    </html>
Wat komt eerder?
Inline CSS heeft altijd de hoogste prioriteit.
Internal en external CSS overschrijven elkaar.
Dat betekent dat de degene die als laatst wordt aangeroepen of geladen wordt de voorkeur krijgt.

Variabelen

Javascript is loosely typed and dynamic


                    let name = "Nick";
                    name = 22;
                    name = true;
Dynamically typed betekent dat de datatype tijdens het uitvoeren van de code wordt bepaald. Een variabele kan dus ook van datatype veranderen. je declareert je variabele dus niet, op het moment dat je de variabele maakt.

Variabele maken

Je kunt op twee manieren een variabele maken:


                    let student = "Rogier Pettendrager";
                    const isMale = true;

What about var?


                    var x = 22; // waarde x is 22
                    {
                        var x = 10; // waarde x is 10
                    }
                    console.log(x);
                    // waarde van variabele x is:  10
                

                    let x = 22; // waarde x is 22
                    {
                        let x = 10; // waarde x is 10
                    }
                    console.log(x);
                    // waarde van variabele x is: 22

Tip: gebruik geen var.

=, == of ===


                    let age = 22;
                    if (age == 22) {
                        console.log("This is true");
                    } else {
                        console.log("This is not true");
                    }
                

                    let age = "22";
                    if (age === 22) {
                        console.log("This is true");
                    } else {
                        console.log("This is not true");
                    }
